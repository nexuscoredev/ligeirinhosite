Configurar login Google

Cole o Client ID do Google Cloud Console. Fica salvo neste navegador.

Como obter o Client ID (5 min)
  1. Abra Google Cloud Console → Credenciais
  2. Crie um projeto (ex.: Ligeirinho App)
  3. Em Criar credenciais → ID do cliente OAuth, escolha Aplicativo da Web
  4. Em Origens JavaScript autorizadas, adicione:
    http://localhost:3456
    https://seudominio.com.br
  5. Copie o ID do cliente e cole acima

Para produção permanente, salve também em data/auth-config.local.json (não vai pro Git).
